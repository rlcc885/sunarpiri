¿Cómo Suscribirme?
Afiliación organizacional en línea

Por medio de esta funcionalidad, cualquier usuario externo pueda afiliarse a la Extranet de SUNARP.

El usuario debe tener una conexión habilitada a Internet, con permiso para acceder a la Extranet.

La secuencia de pasos que el usuario debe seguir es la siguiente:

Empezar Afiliación
1. Ingrese a la página principal de la Extranet
2. De clic sobre el enlace “Deseo Suscribirme” de la sección “Suscripción de Organizaciones”.
Completar Formulario
3. El sistema muestra el formulario de afiliación
4. Ingrese todos los datos solicitados, y de clic sobre el botón “Continuar”.
5. Si el Prefijo de Cuenta ya existe
a. Recibe mensaje de error: prefijo de cuenta especificada ya existe. Para volver a intentarlo, presiona el botón “Regresar” y vuelve a intentarlo

Revisar Contrato
6. Si el Prefijo de Cuenta es aceptado, el sistema le muestra la página web del Contrato de Afiliación

Aceptar Contrato
7. Revise el Contrato de Afiliación y de clic sobre el botón “Imprimir”
8. Si el usuario decide Rechazar el Contrato de Afiliación
a. De clic sobre el botón “No Acepto”
b. Recibe la página de confirmación de la afiliación cancelada
Contrato Aceptado
9. Si el usuario decide Aceptar el Contrato de Afiliación, de clic sobre el botón “Sí Acepto”. El sistema le muestra la página web del Contrato de Afiliación, con los datos ingresados por el usuario

Afiliación Terminada
10. Finalmente, el usuario recibe la página de confirmación de la afiliación satisfactoria
