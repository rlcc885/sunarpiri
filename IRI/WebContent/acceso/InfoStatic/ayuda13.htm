Actualizar el Password

Por medio de esta funcionalidad, el usuario de la Extranet puede actualización de password.

El usuario debe tener una conexión habilitada a Internet, con permiso para acceder a la Extranet.

La secuencia de pasos que el usuario debe seguir es la siguiente:

Seleccionar al Usuario cuyo password desea modificar

1. Ingrese a la página principal de la Extranet
2. Selecciona la opción “Editar mis datos”

Modifique la contraseña

3. Actualiza el campo password.
4. Presiona el botón “Grabar”

Mensaje de Confirmación

5. Se recibe mensaje de confirmación.
